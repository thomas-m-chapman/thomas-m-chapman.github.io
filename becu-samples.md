# Samples of my writing from BECU
The following documents were composed during my last year or two at BECU. 

I wrote these documents when I was serving as Product Owner (PO) for a high-priority effort to replace BECU's existing knowledge base (KB) with a new platform that better met the needs of our internal staff. The existing KB was called InfoPoint. The code name of the new KB was called InfoPoint Next, or IPN.
  * [IPN Content IDs and File Naming Conventions](Tom Chapman - Writing sample - IPN Content IDs and File Naming Conventions.docx) - This document summarizes two key new standards that my Knowledge Management team would need to support when IPN was launched: conventions for Content IDs and File Names.
  * [IPN Content Libraries design document](Tom Chapman - Writing sample - IPN Content Libraries design document.docx) - This document describes the design of a key new feature of IPN known as Content Libraries.
  * [IPN Content libraries and Document types](Tom Chapman - Writing sample - IPN definitions Content libraries and Document types.docx) - This document defines the 13 Content Libraries, 7 document categories, and 46 discrete document types that would be available upon the launch of IPN.
